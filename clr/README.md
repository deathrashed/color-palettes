# clr

Ready-to-use Apple `.clr` color-list files live here.

These are binary `NSColorList` files for macOS color workflows. You can generate or refresh a palette with:

```sh
../bin/colorgen -i '../bin/formats/HTML.json' -o 'HTML Colors'
```

The current collection includes palettes such as:

```text
Ant.Design
Bootstrap
CSS
Chakra UI
Copic
Crayola
Flat UI Colors
HTML Colors
Material Design Colors
Metro Colors
Pantone
Primer
Social Colors
Tachyons
Tailwind
```

Because `.clr` files are binary, inspect them through macOS color tools rather than text diff tools.
